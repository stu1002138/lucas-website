嗨！我叫Lucas.

我是一名全端工程師。

在工作之餘，我喜歡尋找美味的食物和到處旅遊。

偶爾也會分享技術開發與生活旅遊的心得！

我畢業於資訊工程系，目前任職於元大銀行擔任全端工程師，擁有 7 年的開發經驗。

從大學時期的 Android 開發起步，逐步轉向網頁前端，歷經技嘉科技、智誠科技、奧丁丁等公司累積前端經驗後，進一步拓展至全端開發，技術涵蓋 Vue 3 / Nuxt 3 前端與 Java Spring Boot 後端。

工作之餘持續關注前端新技術與後端架構設計，致力於提升程式品質與開發效能。

同時，我也非常喜歡旅遊和品嚐美食。歡迎相同愛好者一起交流！
